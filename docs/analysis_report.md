# [Final Report v3] 리스크 스코어링 기반 금·은 자산배분 분석 리포트

**Date:** 2026-03-21
**Analyst:** Senior Macro Strategy Analyst

## 1. 시장 위험도 지수(Risk Score) 진단
현재 시장의 종합 위험도는 **Low-to-Mid Risk** 구간에 위치해 있습니다 (가중치 설정에 따라 변동 가능). 
- **Vol Factor**: S&P 500의 단기 변동성은 하향 안정화 추세이나, 지정학적 리스크 발발 시 민감도가 급증할 것으로 예상됩니다.
- **Yield Factor**: 10Y Yield의 급격한 변동이 금 가격의 하방 압력으로 작용하는지 실시간 모니터링이 필요합니다.

## 2. GSR Cycle 및 은(Silver) 가치 평가
GSR(Gold-Silver Ratio) 역사적 밴드 분석을 통해 현재 자산 가치를 진단합니다.
- **Insight**: GSR이 역사적 평균 상단(+1.5 SD)에 도달할수록 은(Silver)의 상대적 저평가가 심화됨을 의미하며, 이는 장기적 관점에서 Aggressive 모델로의 스위칭 적기임을 시사합니다.

## 3. 포트폴리오 대응 전략
| 위험도 점수 | 모델명 | 권장 비중 | 핵심 전략 |
| :--- | :--- | :--- | :--- |
| **70점 이상** | **Defensive** | 금 80% / 은 20% | 원금 보존 및 MDD 최소화 (Capital Preservation) |
| **40~70점** | **Balanced** | 금 50% / 은 50% | GSR 회귀를 이용한 중기 수익 확보 |
| **40점 이하** | **Aggressive** | 금 20% / 은 80% | 경기 회복기 산업 수요 탄력성 극대화 |

## 4. KB자산운용 인사이트 접목 결론
과거 COVID-19 및 러-우 전쟁 사례를 분석한 결과, 위기 초기 T+20일까지는 **Defensive** 모델의 방어력이 압도적이었으나, 유동성 공급 및 공포 지수 하락기에는 **Aggressive** 모델의 수익률 탄력성이 금 대비 2.5배 높게 나타났습니다. 따라서 현재의 리스크 스코어를 추세적으로 관찰하며 비중 조절 역동성을 확보할 것을 권고합니다.
